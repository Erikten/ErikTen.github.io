前言

开始之前先来个hello world吧，顺带标记一下php的注释格式：

<?php
    echo "Hello World!";
    //echo "Hello World!";
    #这也是单行注释
    /*
    这是
    多行
    注释
    */
    phpinfo();
?>

变量的写法

变量名：必须以$符开头
变量名组成部分：a-z A-Z 0-9 下划线，但是不能以数字开头！

<?php
    $a = 10;
    echo $a;
?>

变量区分大小写

<?php
    $a = 10;
    $A = 20;
    # 运行会输出10
    echo $a;
?>

在学习变量类型前需要学习三个函数，为了方便我们了解变量类型。

输出变量类型的函数

isset()


现在我们来测试一下：

<?php
    $a = 10;
    $A = 20;
    echo isset($a);
    echo isset($b);
?>


这里可以看到，程序只输出了1，其实它返回了两个值，一个1，一个NULL，为了更好的观察它的返回值，我们在这里使用另一个函数var_dump

var_dump()


现在我们来看一下添加了var_dump()函数后的程序输出怎样的结果：

<?php

    $a = 10;

    $A = 20;

    echo isset($a);
    // echo isset($b);
    var_dump(isset($b))
?>

最后来一个gettype()，查询变量的类型

gettype()

用法：

<?php

    $a = 10;

    $A = 21.2;

    $b = '张三';

    $B = TRUE;

    $c = NULL;
    // echo isset($a);
    // // echo isset($b);
    // var_dump(isset($b))

    echo gettype($a);
    echo '<br>';
    echo gettype($A);
    echo "<br>";
    echo gettype($b);
    echo '<br>';
    echo gettype($B);
    echo '<br>';
    echo gettype($c)
?>

PHP的五个基本数据类型

字符串（string）
  • 字符串要用双引号或者单引号括住，中间可以加html标签
<?php
$number = 10;
echo 'xiaoming jin nian $number sui le ~<br>';
echo "xiaoming jin nian $number sui le ~";
?>

注意：

  1. 用单引号时，程序不会对字符串进行解析，如果包含变量，只会输出变量名，而不会输出变量值
  2. 用双引号时则会对字符串进行解析，如果包含变量，则会对变量进行解析，输出变量值
  3. 当一个字符串是纯文本组成时，尽量用单引号输出
    整数（intger）

例如：

$count = 100;
浮点数（float）

浮点数是带小数点或者指数类型的数字

count = 66.66
布尔型（boolean）

只有两个值TRUE或FALSE 常用于条件测试，例如：

$login = TRUE
NULL值

字面意思，表示为空

<?php
$number = 10;
$login = TRUE;
$count = 3.14;
$null = NULL;
var_dump($count,$login,$null,$number) //打印当前变量类型
?>

PHP的运算符

算数运算符


这个敲一敲就行了，比较简单：

<?php
//定义变量x，y并赋值
$x = 10;
$y = 20;

//进行加减乘除运算并输出结果
echo ($x+$y);
echo '<br>';

echo ($x-$y);
echo '<br>';

echo ($x*$y);
echo '<br>';

echo ($x/$y);
echo '<br>';
?>
字符串运算

在PHP中字符串的拼接用.（点）来完成，例如：

<?php
//定义变量x，y并赋值
$x = 'Hello';
$y = ' world!';
$z = $x.$y;
echo $z;
?>

赋值运算符

示例：

<?php
//定义变量x，y并赋值
$x = 10;
$y = 20;
$z = 5;
$a = 3;
$b = 20;

//开始赋值运算
echo $x+=10;
echo '<br>';
echo $y-=10;
echo '<br>';
echo $z*=10;
echo '<br>';
echo $a/=10;
echo '<br>';
echo $b%=10;
?>
递增递减运算符

递增递减运算符只能作用于变量：

例子：

<?php

    $x = 1;
    $y = 1;

    echo $x++;
    echo '<br>';
    echo ++$y;
?>

这里输出的结果一个为一一个为二，但其实都是2，之不过一个先输出再自加，一个先自加再输出而已，有点类似于二进制里边的压栈出栈的感觉，顺序不同而已。

比较运算符


这里强调说一下等于（==）和全等于（ ===）的区别：

<?php
    $x = 123;
    $y = '123';


    if($x == $y)
    {
        echo '$x dengyu $y';
    }
    if($x === $y)
    {
        echo '$x quandengyu $y';
    }
?>

运行结果：

可以看到，全等于不仅要求数值相同，数据结构也得一样，x是整型，y是字符串，所以不会输出结果

PHP中的控制结构

选择结构（if、switch）

这里的if-else跟其他编程语言的没啥区别，就是写法不同：

<meta charset="utf-8">
<?php

    $a = 10;
    $b = 20;

    if ($a>5)
    {
        echo 'a>5';
    }
    elseif ($a>1)
    {
        echo "a>1";
    }
    else
    {
        echo "不符合条件";
    }
?>

接着就是switch语句，switch后跟一个变量，case后跟条件，符合就break，不符合顺序执行下一条case，都不符合就执行default：

<meta charset="utf-8">
<?php

    $a = 10;
    $b = 20;

    switch ($a)
    {
        case 1:
            echo "a = 1";
            break;
        case 2:
            echo "a = 2";
            break;
        case 6:
            echo "a = 6";            
            break;
        default:
            echo "a = ???";
            break;
    }
?>
循环结构（while、for、do-while）

while后跟条件，中间是循环体，这是while在编程语言中的典型用法：

<meta charset="utf-8">
<?php

    $a = 10;
    $b = 20;

    while ($a<$b)
    {
        echo "a = $a,b = $b".'<br>';
        $a++;
    }
?>

运行结果：

接着是for，这里的for循环跟C语言中的for循环有点相似，先定义一个初始变量i，跟上一个循环条件，最后做一个自变来防止死循环：

<meta charset="utf-8">
<?php

    for ($i=0; $i < 100; $i+=2) { 
        # code...
        echo "i = $i"."<br>";
    }

?>


最后是do-while，它跟while的区别就是，do-while是先执行语句，在判断循环，while则是先判断循环，在执行语句：

<meta charset="utf-8">
<?php
    $a = 15;
    $b = 10;
    do
    {
        echo "a =$a".'<br>';
        $a--;
    } while ( $a>$b);

?>

输出九九乘法表

重点在于两个for循环的嵌套，i必须小于10，并且自加，j必须小于等于i，也进行自加，且它俩初始值都为0

<?php
for ($i=0; $i <10 ; $i++) 
{ 
    for ($j=0; $j <=$i ; $j++) { 
        echo "$i*$j=".($i*$j).'&nbsp'.'&nbsp'.'&nbsp';
    }
    echo "<br/>";
}
?>
