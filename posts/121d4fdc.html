这里会用到几个重要的函数去操作数据库，连接、增删改查，这些操作，我会在下面一一列举出来的，最后的实战一定要去写一下，这样才能明白PHP究竟是怎么操作Mysql的，因为是为了实现效果，所以没有做安全优化，我也是第一次写，代码可能会有些不严谨。

mysqli_query()

它的作用是执行SQL语句，格式：

mysqli_query($link,'select * from database')

例子：


<?php 
// 假定数据库用户名：root，密码：123456，数据库：RUNOOB 
$con=mysqli_connect("localhost","root","123456","RUNOOB"); 
if (mysqli_connect_errno($con)) 
{ 
    echo "连接 MySQL 失败: " . mysqli_connect_error(); 
} 

// 执行查询
mysqli_query($con,"SELECT * FROM websites");
mysqli_query($con,"INSERT INTO websites (name, url, alexa, country)
VALUES ('百度','https://www.baidu.com/','4','CN')");

mysqli_close($con);
?>

mysqli_fetch_assoc()

它的作用是从结果集中取得一行作为关联数组，格式：

mysqli_fetch_assoc(result);

例子：


<?php 
// 假定数据库用户名：root，密码：123456，数据库：RUNOOB 
$con=mysqli_connect("localhost","root","123456","RUNOOB"); 
if (mysqli_connect_errno($con)) 
{ 
    echo "连接 MySQL 失败: " . mysqli_connect_error(); 
} 

$sql="SELECT name,url FROM websites ORDER BY alexa";
$result=mysqli_query($con,$sql);

// 关联数组
$row=mysqli_fetch_assoc($result);
printf ("%s (%s)\n",$row["name"],$row["url"]);

// 释放结果集
mysqli_free_result($result);

mysqli_close($con);
?>

网页实现增删改查

首先要知道增删改查的命令：

# 增加
insert into student(name,age,classID) value('$name',$age,$county);
# 删除
delete from student where id=$id;
# 修改
update student set name=$name,age=$age,classID=$county where id=14;
# 查找
select student.id,student.name,student.age,class.clasName from student left join class on student.classID=class.id;
实现查找

<?php

    {
